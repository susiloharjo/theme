### Local Development

1.  **Install Dependencies**
    ```bash
    npm install
    ```

2.  **Run Development Server**
    ```bash
    npm run dev
    ```
    Open `http://localhost:4200` (or the port shown in terminal) in your browser.

3.  **Build for Production**
    ```bash
    npm run build
    ```
    Artifacts will be in `dist/eris-app`.
